# study
### <卒業研究の説明>
背景：私自身が基本情報技術者試験を受験した際、シラバス改定などに伴う新出単語を十分に学べる教材が不足していることに問題意識を持ちました。日々更新されるIT技術の学習においては、正しい知識を迅速に提供することが求められるため、新たな学習教材には情報の「正確性」と対応の「即時性」を両立させることが不可欠であると考えています。  
目的：学習者が新出単語を効果的に学習できるようにすること  
方法：LLMを用いた問題作成(AQG)、自動採点・フィードバック評価を行うようにする  
成果の目標：システムを用いない場合と比較してシステムを用いることで素早く正確に学習教材を作成できるようにし、学習者がそのシステムを利用することによって新出単語の学習率が向上すること  

### <修士研究の説明>
背景：卒業研究の結果を踏まえて  
目的：学習者が新出単語を効果的に学習できるようにすること  
方法：LLMを用いた問題作成(AQG)、自動採点・フィードバック評価を行うようにする  
成果の目標：システムを用いない場合と比較してシステムを用いることで新出単語の学習率が向上すること
